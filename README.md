# 🏠 Airbnb Tokyo Analytics Platform

An end-to-end data analytics and AI platform for exploring the Tokyo Airbnb market.

## 🚀 Features

- **📈 Interactive Dashboard** - Key metrics, price distributions, and neighborhood analysis
- **🤖 AI Assistant** - Natural language Q&A using LangChain and OpenAI
- **🗺️ Interactive Maps** - Geospatial clustering of listings
- **💬 Sentiment Analysis** - Guest review sentiment analysis
- **🔮 Price Predictor** - ML-based price prediction tool
- **📊 Data Explorer** - Raw data browsing with filters
- **📥 Export Reports** - Download CSV reports

## 🛠️ Tech Stack

- **Frontend**: Streamlit
- **AI/ML**: LangChain, OpenAI, scikit-learn
- **Data**: Pandas, NumPy, GeoPandas
- **Visualization**: Matplotlib, Seaborn, Folium
- **Deployment**: Hugging Face Spaces

## 📊 Data Source

[Inside Airbnb Tokyo](https://insideairbnb.com/get-the-data/)
